# Exos_Arm
It's a little fun project which is about some kind of exoskeleton forearm, hence the name Exos-Arm.
It's just a way of keeping myself busy and i have to admit that i like to build stuff from scratch.
The 3 main parts of the project are:

- Mechanics through CAD and 3D printing

 ![CAD](https://github.com/MECANICVS/Exos_Arm/assets/83305511/e1e38543-36ee-498e-8186-e3b89d50a4cf)
 
- Electronics with an EMG(Electromyogram) which i'm still working on

 ![schematic](https://github.com/MECANICVS/Exos_Arm/assets/83305511/ee17defe-89e6-4080-a9c7-2d282bcb8859)
 
- Bare-metal Embedded Software with a Nucleo Board https://youtu.be/MnuAzRUYLBU

This is just a very rough prototype of what it is supposed to look like, i'll improve it later on by
downsizing the parts, and finishing the EMG. I also think of implementing an RTOS version of the code.
