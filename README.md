# Exos_Arm
It's a little fun project which is about some kind of exoskeleton forearm, hence the name Exos-Arm.
It's just a way of keeping myself busy and i have to admit that i like to build stuff.
The 3 main parts of the project are:

- Mechanics through CAD and 3D printing
 
- Electronics with an EMG which i'm still working on

- Bare-metal Embedded Software with a Nucleo Board

This is just a very rough prototype of what it is supposed to look like, i'll improve it later on by
downsizing the parts, and finishing the EMG. I also think of implementing an RTOS version of the code.
